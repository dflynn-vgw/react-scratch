# Documentation

This directory contains documentation and assets for the React Scratch project.

## Structure

```
docs/
├── images/           # Screenshots and visual assets
│   ├── home.png     # Home page screenshot
│   ├── about.png    # About page screenshot
│   └── mobile.png   # Mobile responsive view
├── SCREENSHOTS.md   # Guide for capturing and adding screenshots
└── README.md        # This file
```

## Screenshots

The `images/` directory contains screenshots used in the main README.md file. These screenshots demonstrate the application's features and responsive design.

### How to Add Screenshots

1. See [SCREENSHOTS.md](./SCREENSHOTS.md) for detailed instructions
2. Screenshots should be PNG format for best quality
3. Follow the naming conventions specified in the guide

### Required Screenshots

- **home.png** - Screenshot of the Home page showing the welcome message and sanitation services theme
- **about.png** - Screenshot of the About page demonstrating the responsive grid layout
- **mobile.png** - Screenshot showing mobile/responsive view (375px or similar mobile width)
